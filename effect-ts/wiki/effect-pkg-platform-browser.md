# @effect/platform-browser

Browser-targeted implementations of Effect's platform services. Provides Fetch and XHR HTTP clients, WebSocket sockets, `localStorage`/`sessionStorage` KeyValueStores, Clipboard, Geolocation, Permissions, browser Worker/SharedWorker platforms, DOM event streams, a top-level `runMain`, and a schema-driven IndexedDB module with typed migrations. Designed to be imported from browser bundles and layered together with `effect` core.
