# KYC/AML para Intermediários de Comércio

## Marco Legal

- **Lei 9.613/1998** — Lei de Anti-Lavagem de Dinheiro; estabelece COAF
- **Lei 12.683/2012** — Ampliou escopo para entidades não-financeiras (intermediários de comércio)
- **Circular Banco Central 3.978/2020** — Abordagem baseada em risco para instituições financeiras
- **Resoluções CMN 29/2017 & 25/2019** — Requisitos atualizados de KYC/CDD

## Obrigações para Intermediários de Comércio

| Obrigação | Detalhe |
|------------|--------|
| Identificação de Cliente (KYC) | Verificar identidade de clientes incl. beneficiários finais |
| Due Diligence de Cliente (CDD) | Compreender propósito e natureza do negócio |
| Due Diligence Aprimorada (EDD) | Necessária para PEPs, regiões sancionadas, padrões incomuns |
| Monitoramento de Transação | Sistemas para detectar padrões suspeitos |
| Denúncia de Atividade Suspeita (SAR) | Reportar ao COAF dentro de **24 horas** |
| Reporte de Transação Grande | ≥ R$10.000 (certas contas); R$100.000 (estrangeira) |
| Guarda de Registros | **Mínimo 5 anos** para todos os registros |
| Rastreamento de PEP | Rastrear contra listas de PEPs + sanções internacionais |

## Riscos Específicos China-Brasil

- **Subfaturamento**: Facilitar embarques com declarações subavaliadas = evasão fiscal + violação AML
- **Exportadores de empresa fantasma**: Estruturas corporativas chinesas opacas exigem due diligence aprimorada
- **Operações com muito dinheiro**: Fluxos de caixa não documentados são bandeiras vermelhas
- **Risco de sanções**: Entidades chinesas em listas de sanções EUA/UE criam exposição

## Programa Prático de Conformidade

1. Documentar padrões KYC, apetite por risco, caminhos de escalação
2. Onboarding de cliente: Verificação CNPJ, estrutura corporativa, beneficiários finais
3. Monitoramento contínuo: Sinalizar valores de pedido incomuns, mudanças rápidas de volume
4. KYC de fornecedor: Verificar fornecedor chinês via históricos Alibaba, certificações, auditorias fabris
5. Designar oficial de conformidade AML
6. Treinamento regular da equipe
7. Registrar com COAF se não supervisionado por outra autoridade
