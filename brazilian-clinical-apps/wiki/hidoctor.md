HiDoctor® is a Brazilian medical software platform developed by **Centralx®**, a company with 35+ years of experience in medical technology. HiDoctor claims 75,000+ health professionals, 380 million+ scheduled appointments, and 90 million+ patients served — making it one of the largest platforms by stated user scale in the Brazilian market alongside [[amplimed]] and [[prodoctor]]. Its principal differentiator is being the **only multi-platform medical system in Brazil**: it operates simultaneously as a desktop-installed client (with offline functionality), a web-classic version, a web-internet version, and native mobile apps (iOS and Android), all sharing the same patient data in real time. This offline capability sets it apart from fully cloud-based competitors like [[feegow]], [[consultorio-live]], and [[simdoctor]].

## Key Features

HiDoctor is distinguished by its multi-platform architecture and an unusually broad module set:

| Feature | Description |
|---|---|
| **HiDoctor LIVE (AI)** | AI clinical assistant: transcribes and summarizes consultations, generates real-time clinical insights, suggests questions and examination orders |
| **Multi-Platform Access** | Desktop (offline-capable, installed), Web Classic, Web Internet, Mobile App — same data across all platforms simultaneously |
| **Offline Mode** | Full EHR and scheduling functionality without an internet connection — unique in the Brazilian market |
| **Prontuário Personalizado** | Customizable EHR templates per specialty; import from reference physician templates |
| **Telemedicina** | Full teleconsultation with integrated EHR and legally valid digital prescription |
| **Assinatura Digital** | ICP-Brasil batch digital signing of records |
| **MedBook** | Patient relationship platform: send medical documents, enable online scheduling, patient app |
| **HiDoctor GS (Financial)** | Integrated billing, procedure/materials tracking, automatic repasse calculation, inventory, [[tiss]] |
| **Marketing Module** | WhatsApp/email/SMS automated messages with sophisticated patient filters |
| **Confirmação Automática** | WhatsApp/SMS appointment reminders; patient can confirm/cancel directly |
| **Agenda** | Organized scheduling accessible from any platform; separate secretary-access app |
| **App do Consultório** | Dedicated clinic app with online scheduling and patient communications |
| **CatalogoMed** | Integrated drug catalog and clinical reference |
| **Atlas de Anatomia** | Anatomy atlas embedded in the platform |
| **Faturamento TISS / TUSS** | Health plan billing; XML generation; guide printing |
| **Bulário** | Drug reference database |
| **Chat Interno** | Internal clinic team chat |
| **Relatórios e Indicadores** | Revenue, productivity, and profitability dashboards |
| **Sincronia Multiponto** | Multi-device real-time synchronization |
| **Site Médico** | Professional website for the clinic or practice |

## Target Market

- Medical clinics and consultórios across all specialties
- Specialties specifically highlighted: Cirurgia Plástica, Endocrinologia, Oftalmologia, Pediatria & Nutrologia, Ginecologia & Obstetrícia, Medicina Reprodutiva, Dermatologia, Mastologia
