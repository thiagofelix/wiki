Consultório Live is a cloud-based medical clinic management platform developed by **VBB Software Ltda**, a Brazilian software company active in the clinic/consultório software space since **2004**. It is browser-based (no installation required) and targets health and wellness professionals across all specialties and practice sizes, from a newly opened solo practice to a large polyclinic with multiple branches and staff. Its key competitive positioning is **transparent, low entry pricing** (plans from R$ 59/month for 1 user), making it accessible to solo practitioners — contrasting with premium-tier platforms like [[amplimed]] or [[hidoctor]]. VBB Software also offers a companion **Consultório Desktop** product for practices preferring an offline-installed Windows client.

## Key Features

| Feature | Description |
|---|---|
| **Agenda Online** | Per-professional scheduling; encaixes (walk-in slots), recurring appointments, schedule locking |
| **Confirmação Automática** | WhatsApp, SMS, email automatic confirmations; patient can confirm or cancel directly |
| **Sala de Espera Virtual** | Digital waiting room with real-time patient arrival tracking; time-in-queue visibility for staff |
| **Painel de Chamada** | Patient call display on clinic TV screen; room routing |
| **Agendamento Online** | Free clinic website with embedded patient self-scheduling; embeds into existing site |
| **Prontuário Eletrônico** | Complete patient registration with photos and attachments; custom anamnesis models |
| **Linha do Tempo** | Consultation timeline with filters and evolution graphs |
| **Teleconsulta** | Video calls integrated with patient record |
| **Prescrição Digital** | Integrated with **Memed** (60,000+ drug database); ICP-Brasil digital signature |
| **Gestão Financeira** | Cash flow, accounts payable/receivable, receipts with auto-fill, [[tiss]]/ANS XML |
| **TISS / ANS** | Guide generation (Consulta, SP/SADT), XML lot export for health plan operators |
| **Marketing** | Email marketing integration (MailChimp); post-consultation patient evaluation/review collection |
| **IA Summary** | AI-generated intelligent summaries of patient evolution (new feature) |
| **2-Factor Authentication** | Two-factor authentication for account security |
| **Estoque** | Product inventory with lot numbers, expiry dates, stock movement alerts |
| **API** | Open API for developer integrations |
| **Mobile App** | iOS and Android apps for agenda and patient record access |
| **SSL 256-bit** | Enterprise-grade encryption; AWS-hosted; [[lgpd-health-data]] compliant |

## Tech Stack

- **100% cloud-based** (browser-based; no client installation for the main product)
- Hosted on **AWS**; **SSL 256-bit** encryption
- Compatible: Windows, Mac, Linux, iOS, Android (via browser + native apps)

- Also: **Consultório Desktop** companion product (Windows-installed, offline-capable)
